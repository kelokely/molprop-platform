MolProp Platform: web + advanced analytics on top of MolProp Toolkit tables.

This repo is the fast-moving companion to molprop-toolkit. The toolkit stays stable and owns the results-table schema; the platform adds interactive apps (Streamlit), visualization dashboards (Plotly), and higher-level medchem workflows (MMP/SAR/Pareto/lookup/bioisosteres).

Quickstart Core toolkit

Install

Recommended (dev + viz)

git clone https://github.com/kelokely/molprop-platform.git
cd molprop-platform

pip install -e ".[dev,viz]"

Installs the platform plus visualization dependencies. Core toolkit integration is an extra.

With core toolkit integration

# installs molprop-toolkit from GitHub (until core is on PyPI)
pip install -e ".[core]"

# recommended full setup
pip install -e ".[core,web,viz]"

Enables Streamlit UI and reads MolProp results tables produced by the core toolkit.
