Quickstart Tools GitHub

Run MolScope from one workspace.

Upload files, run the current MolScope commands, inspect outputs, and download the full bundle when you are done. The toolkit still owns the chemistry table and schema. This repo gives that workflow a server and a clean browser-facing workbench.

Get started Toolkit docs

One workspace stays open for the whole run. You can upload SMILES, results tables, or bundle zips, run commands in place, preview tables, and keep the logs and outputs together.

Generate

Start from .smi, build a results table, and keep going into report, picklists, and visualization.

Work from a table

Run compare, SAR, MMP, search, similarity, featurize, retro, learnings, dashboard, and portal workflows from the same place.

Install
git clone https://github.com/kelokely/molprop-platform.git
cd molprop-platform

pip install -e ".[dev,web,viz]"
What you get

The same install brings in the server workspace, the projection tools, and the MolScope toolkit command surface.

Start the Streamlit workspace.

molscope-server

Inside the app you can add files, run commands, inspect previews, and download the full run directory as one zip.

Use the lightweight visualization command directly when you want a quick projection outside the server.

molscope-visualize results.parquet -o viz --method umap

molscope-mmp and molscope-sar also forward to the toolkit commands when the toolkit is installed.

The toolkit remains the source of truth for the chemistry table and command language. This server sits on top of that surface instead of replacing it. 

molscope calc input.smi -o results.parquet
molscope compare reference.parquet candidate.parquet
molscope portal --report-dir report --picklists-dir picklists
What the server can run

calc, prep, analyze, report, picklists, integrate, compare, SAR, MMP, search, series, similarity, featurize, retro, learnings, dashboard, portal, and schema checks.

What stays with the toolkit

Chemistry standardization, schema rules, and the underlying command implementations.
