Quickstart

Use this when you want the server workspace up quickly and you want the full MolScope workflow available inside it.

1) Install

git clone https://github.com/kelokely/molprop-platform.git
cd molprop-platform

pip install -e ".[dev,web,viz]"
pip install -e ".[core]"

The first line installs the server and projection stack. The second line brings the MolScope toolkit into the same environment.

2) Start the server

molscope-server

The app opens one workspace for the run. You can keep adding files and outputs to that same workspace.

3) Run a simple path

Add a .smi file in the Workspace tab, then open Generate and run the default pipeline.

That gives you a results table, optional report and picklists, and an optional quick projection in the same run folder.

4) Run table workflows

Once a results table is in the workspace, the Commands tab can run compare, SAR, MMP, search, similarity, featurize, retro, learnings, dashboard, portal, and schema checks from the same place.

5) Quick CLI projection

molscope-visualize path/to/results.parquet -o viz --method umap

This writes a projection CSV and an interactive HTML view.
