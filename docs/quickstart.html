Quickstart

This is a minimal end-to-end run: install the platform with visualization extras, then generate a PCA plot from a MolProp results table.

1) Install

git clone https://github.com/kelokely/molprop-platform.git
cd molprop-platform

pip install -e ".[dev,viz]"

If you want the Streamlit UI too, install:

pip install -e ".[web]"

# and to pull the core toolkit from GitHub
pip install -e ".[core]"

2) Run visualization

Point the tool at an existing MolProp results table (CSV/TSV/Parquet) and it will write an HTML scatter plot. 

molprop-visualize path/to/results.parquet -o viz --method pca

3) Open output

open viz/projection_pca.html
