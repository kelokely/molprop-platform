CLI tools

All tools are table-first. Optional dependencies are behind extras such as [viz] and [web].

molprop-web

Starts the Streamlit UI. Supports both SMILES → table (when core toolkit is installed) and table → artifacts workflows.

molprop-visualize

Creates a PCA/UMAP projection from a results table and writes an interactive HTML scatter plot plus projection CSV.
