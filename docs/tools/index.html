Tools

The server keeps the command language close to the toolkit. Most of the heavy lifting still runs through the MolScope toolkit in the same environment.

molscope-server

Starts the server workspace. Use it to upload files, run commands, inspect outputs, and download the full run directory.

molscope-visualize

Builds a PCA or UMAP projection from a MolScope table and writes an HTML view plus a projection CSV.

molscope-mmp

Forwards to the toolkit MMP workflow when the toolkit is installed in the same environment.

molscope-sar

Forwards to the toolkit SAR workflow when the toolkit is installed in the same environment.
